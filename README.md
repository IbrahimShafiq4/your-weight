# Your Weight on Every Planet

Welcome to the "Your Weight on Every Planet" project! Ever wondered how much you would weigh on other planets in our solar system? This project allows you to input your weight and see how it compares on different celestial bodies - it's a fun way to learn about the differences in gravity across our solar system!

- [Demo](#demo)

## Introduction
The "Your Weight on Every Planet" project provides users with an educational tool to understand how gravity affects weight on various planets. Input your weight, and the application will calculate and display your weight on each planet, giving you a unique perspective on how gravity varies throughout our solar system.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/IbrahimShafiq4/your-weight.git
   ```
2. Navigate to the project directory:
   ```bash
   cd your-weight
   ```
3. Open `index.html` in your web browser to launch the "Your Weight on Every Planet" website.

## Usage
1. Input your weight in the provided field.
2. Press the "Calculate" button to see your weight on every planet.
3. Explore and compare your weight on different celestial bodies in our solar system!
